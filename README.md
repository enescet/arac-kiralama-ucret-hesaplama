# Araç Kiralama Ücret Hesaplama (Android)

Bu proje, Android Studio kullanılarak geliştirilmiş basit bir araç kiralama ücret hesaplama uygulamasıdır.

## Uygulama Özellikleri
- Kiralama gün sayısına göre hesaplama
- Araç sınıfı seçimi (Eco, VIP, SUV)
- Vites türü seçimi (Manuel / Otomatik)
- Ek hizmetler (Şoför, Sigorta, GPS)
- Günlük ve toplam ücret hesaplama

## Kullanılan Teknolojiler
- Android Studio
- Java
- XML
- Minimum SDK: API 21

## Kurulum
1. Projeyi indirin veya klonlayın
2. Android Studio ile açın
3. Gradle senkronizasyonunu tamamlayın
4. Emülatör veya gerçek cihazda çalıştırın

## Ekran Görüntüleri
Uygulamanın çalışır hâline ait ekran görüntüleri proje raporunda yer almaktadır.

## Açıklama
Bu proje, Mobil Programlama dersi kapsamında geliştirilmiştir.
